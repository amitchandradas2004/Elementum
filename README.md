# Elementum Landing Page

A clean and modern landing page UI built using **HTML**, **CSS**, and **Tailwind CSS**.  
This project focuses on layout structure, typography, and responsive design principles.

---

## 🚀 Features

- Responsive design for all devices
- Modern and minimal UI
- Clean typography and spacing
- Reusable Tailwind CSS utility classes
- Well-structured layout sections

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Tailwind CSS
- Daisy UI
